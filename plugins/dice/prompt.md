# /dice

결정을 도와주는 주사위.

## 사용법

```
/dice                    # 1-6 사이 숫자
/dice 20                 # 1-20 사이 숫자 (D&D 스타일)
/dice 피자 치킨 햄버거    # 셋 중 하나 선택
/dice 할까 말까          # 둘 중 하나
```

## 동작

1. 인자가 없으면: 1-6 주사위
2. 숫자 하나면: 1-N 주사위
3. 여러 단어면: 그 중 하나 랜덤 선택

## 출력 형식

```
🎲 [결과]
```

한 줄로 끝. 설명 없이. 결과만.

## 예시

입력: `/dice 재택 출근`
출력: `🎲 재택`

입력: `/dice 100`
출력: `🎲 73`
