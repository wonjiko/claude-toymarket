# ppt-designer

topic-outline.md를 기반으로 HTML 프레젠테이션을 생성하는 플러그인.

## 왜 존재하는지

PowerPoint나 Google Slides 없이 마크다운 기반 아웃라인만으로 발표 자료를 빠르게 만들기 위해. HTML로 출력하므로 별도 소프트웨어 불필요.

## 언제 쓰면 안 되는지

- 기업 표준 템플릿(pptx)이 필요한 경우
- 복잡한 애니메이션이나 차트가 필요한 경우
- 인쇄용 슬라이드가 필요한 경우

## 구성

- `skills/ppt-designer/` - 프레젠테이션 생성 skill
